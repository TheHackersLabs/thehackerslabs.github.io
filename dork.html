Google Dorking: Professional Recon

DevOps & Secretos de Código
Exposición de Directorios .git Crítico inurl:"/.git/config" -github -gitlab

Si el servidor web permite listar el directorio .git, un atacante puede descargar el historial completo del código fuente usando herramientas como 'git-dumper'.

Configuración de Docker & Compose "docker-compose.yml" intext:"services:" "image:"

Revela la arquitectura interna de contenedores, nombres de imágenes y, a menudo, variables de entorno con credenciales.

Ficheros de Entorno (.env) filetype:env "DB_PASSWORD" | "AWS_ACCESS_KEY_ID"

El "Holy Grail" del dorking. Contiene llaves de acceso a bases de datos, APIs de pago (Stripe) y servicios cloud.

Hunting de Vulnerabilidades (SQLi/XSS)
Búsqueda de SQL Injection (Error Based) inurl:id= & intext:"sql syntax error" | "mysql_fetch_array"

Busca páginas que devuelven errores de sintaxis SQL directamente al navegador, lo que indica que los parámetros no están sanitizados.

Parámetros de Redirección (Open Redirect) inurl:redirect= | inurl:url= | inurl:next= "http"

Identifica puntos donde un atacante puede redirigir a un usuario a un sitio malicioso manteniendo la confianza en el dominio original.

Exposición de phpInfo() ext:php intitle:phpinfo "published by the PHP Group"

Revela versiones exactas de PHP, módulos cargados, rutas del sistema y variables de entorno del servidor.

Infraestructura Corporativa & CMS
Consolas SAP & NetWeaver intitle:"SAP NetWeaver Portal" inurl:"/irj/portal"

Puerta de entrada a sistemas ERP. A menudo vulnerables si no están actualizados o usan credenciales por defecto.

Directorios wp-content (WordPress) inurl:wp-content/uploads/ "index of"

Permite navegar por archivos subidos por usuarios. A menudo contiene documentos PDF olvidados o imágenes sensibles.

Sistemas de Videoconferencia (WebEx/Zoom) inurl:zoom.us/j/ "password="

Busca enlaces a reuniones indexados por error que incluyen la contraseña de acceso en la propia URL.

Documentación Interna & PII
Listas de Contactos y VCards filetype:vcf "BEGIN:VCARD"

Extrae información de contacto personal indexada desde servidores de correo mal configurados.

Facturas y Pagos (PDF) filetype:pdf "Invoice" "Confidential" "Paid"

OSINT sobre proveedores y flujos de caja de una organización.

Logs de Transacciones Bancarias filetype:log "SWIFT" | "IBAN" | "Transfer"

Extremadamente sensible. Revela transferencias financieras si los logs son públicos.

Shodan: Banner & Network Intelligence

Filtros Técnicos Avanzados
Búsqueda por ASN (Sistema Autónomo) asn:AS15169

Muestra todos los activos pertenecientes a un ISP o empresa específica (ej. AS15169 es Google).

Filtro de Certificados SSL (CN) ssl.cert.subject.cn:"target.com"

Encuentra servidores que no están en el DNS público pero que tienen instalado un certificado para el dominio objetivo.

Hash de Favicon (Tracker) http.favicon.hash:-1507567067

Busca el icono de la web. Útil para encontrar todas las instancias de una tecnología (ej. Jira) aunque hayan cambiado el nombre del título.

Infraestructura Crítica (SCADA) Industrial
Protocolo BACnet (Control de Edificios) port:47808 "BACnet"

Sistemas de aire acondicionado, iluminación y acceso físico en edificios inteligentes.

Sistemas Niagara Fox "Niagara Fox" port:1911,4911

Framework muy común en la gestión de infraestructuras industriales y de energía.

Terminales de Gasolineras (Guardian) "Tank Inventory" port:10001

Lectura remota de niveles de combustible de estaciones de servicio.

Hardware & Networking
Routers MikroTik (WinBox) port:8291 "MikroTik"

Busca consolas de gestión de MikroTik. A menudo atacadas por vulnerabilidades en WinBox.

VPNs Fortigate / Fortinet "Fortigate" http.title:"FortiToken Clock Drift"

Identifica puntos de entrada VPN. Útil para buscar CVEs específicos de Fortinet.

Servidores de Impresión HP JetDirect product:"HP JetDirect" port:9100

Las impresoras suelen ser el eslabón más débil. Se puede extraer historial de impresión y documentos.

Databases & Cloud Apps
Redis sin Contraseña Crítico port:6379 "redis_version"

Redis expuesto permite ejecución de comandos remotos (RCE) en ciertas configuraciones.

Dashboards de Kubernetes "Kubernetes Dashboard" port:443

Si no requiere autenticación, da control total sobre el clúster de microservicios.

Instancias de Docker API port:2375 "Docker"

Permite desplegar o borrar contenedores directamente si el puerto está expuesto sin TLS.

Metodología: El Enfoque del Atacante

Lógica de Encadenamiento Avanzado

El recon profesional no usa comandos sueltos, usa flujos de inteligencia. Aquí tienes un ejemplo de cómo combinar ambos mundos:

  1. Shodan: Encuentra IPs de una empresa filtrando por certificado:
    ssl.cert.subject.cn:"empresa.com"
  2. Shodan: Identifica cuáles tienen una tecnología específica (ej. Nginx antiguo):
    org:"Empresa X" product:"nginx" version:"1.14.0"
  3. Google: Busca archivos de configuración olvidados en esos subdominios encontrados:
    site:sub.empresa.com ext:bak | ext:old | ext:txt
  4. Google: Localiza si el código fuente de esos servidores está en GitHub con leaks:
    site:github.com "empresa.com" "API_KEY"
